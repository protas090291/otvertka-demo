# Стратегическая презентация «Отвёртка»

Лендинг-презентация: не только описание продукта, а **стратегическое планирование на год**, риски в развитии, рост программы и компании. Цвета и стиль — как у основного приложения. Запускается отдельно от программы.

## Запуск (рекомендуется)

Из папки **Helper2** одной командой:

```bash
cd Helper2
npm run landing
```

Скрипт сам соберёт лендинг, запустит сервер на порту **5175** и откроет браузер. Сервер работает, пока открыт терминал (остановка — **Ctrl+C**).

## Только разработка (без открытия браузера)

```bash
cd Helper2/landing
npm run dev
```

## Только раздача уже собранной версии

```bash
cd Helper2
npm run landing:build
npm run landing:serve
```

## Редактирование контента

- Тексты, контакты и блоки редактируются в `src/App.tsx`.
- Стили (цвета, шрифты) — в `src/index.css` и `tailwind.config.js`.
- Цветовая гамма и стиль совпадают с основным приложением (slate, синие акценты, стеклянные карточки, Inter).
