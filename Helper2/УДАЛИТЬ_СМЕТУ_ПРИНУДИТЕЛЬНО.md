# Удалить смету принудительно (когда интерфейс завис)

Если после загрузки сметы всё лагает и кнопки не нажимаются, удалите данные напрямую в Supabase:

1. Откройте **Supabase Dashboard**: https://supabase.com/dashboard  
2. Выберите ваш проект (тот же, что в приложении).  
3. В левом меню откройте **SQL Editor**.  
4. Нажмите **New query**.  
5. Откройте в проекте файл **`supabase_estimate_clear_data.sql`**, скопируйте его содержимое целиком и вставьте в окно запроса.  
6. Нажмите **Run** (или Ctrl+Enter).

После этого все позиции сметы и записи смет будут удалены из базы. Обновите страницу приложения (F5) — смета будет пустой, интерфейс должен снова нормально работать.
